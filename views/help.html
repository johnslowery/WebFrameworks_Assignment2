Method URL Description
GET / This page, which is a help page that shows all available paths
GET /Users Returns a list of any created users as a JSON array with all properties (UserID, First Name, Last Name, Email Address, Password)
POST /User Can create a new User with the correct properties. It will also check if the inputted UserID has already been used in the User array where it will throw a 404 Error
GET /User/{ID} Should return a specific User, based on their UserID, including all of their properties. If this User does not exist, it will throw a 404 Error
PATCH /User/{ID} Can update the properties (other than UserID) of an inputted User. If the inputted User does not exist, it will throw a 404 Error
DELETE /User/{ID} Can delete a specified User from the User Array. If the inputted User does not exist, it will throw a 404 Error
Any Any request that has not been outlined above Will redirect to this page
